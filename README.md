# Proyecto Final de Telecomunicaciones I

Autor: Luis Morel

## Descripción

Este repositorio contiene los flowgraphs, scripts y documentación del
proyecto final desarrollado con GNU Radio.

## Contenido

- Task 1: Receptor FM con RTL-SDR
- Task 2: Transmisión de video mediante GMSK y canal AWGN
- Task 3: Transmisión GMSK utilizando HackRF y RTL-SDR
- Task 4: Sistema ATSC
- Task 5: Transmisión OFDM
- Task 6: Módem QAM mediante TCP y SDR

## Requisitos

- GNU Radio 3.10.12
- Python 3
- RTL-SDR
- HackRF
- VLC o FFmpeg para las pruebas de video

## Ejecución

Cada carpeta contiene su archivo `.grc` y los scripts necesarios para
realizar la prueba correspondiente.

## Informe

El informe final se encuentra en la carpeta `Informe`.
